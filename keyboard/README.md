# Keyboard Layout

Custom Dvorak-based XKB layout with Swedish characters and programming symbols.

## Base Layer

```
┌───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┐
│ 1 │ 2 │ 3 │ 4 │ 5 │ 6 │ 7 │ 8 │ 9 │ 0 │ + │ § │
├───┼───┼───┼───┼───┼───┼───┼───┼───┼───┼───┼───┤
│ ' │ , │ . │ p │ y │ f │ g │ c │ r │ l │ / │ ¨ │
├───┼───┼───┼───┼───┼───┼───┼───┼───┼───┼───┴───┤
│ a │ o │ e │ u │ i │ d │ h │ t │ n │ s │ -     │
├───┼───┼───┼───┼───┼───┼───┼───┼───┼───┴───────┤
│ ; │ q │ j │ k │ x │ b │ m │ w │ v │ z         │
└───┴───┴───┴───┴───┴───┴───┴───┴───┴───────────┘
```
